# D&D PG Manager

A simple web application to manage Dungeons & Dragons 3.5 edition characters. This app helps with tracking character stats, HP, experience, and level-ups, reducing the need for paper character sheets.

## How to Use

1.  Open the `index.html` file in a web browser.
2.  Click "➕ Aggiungi PG" to create a new character.
3.  Fill in the character details and save.
4.  The character will appear on the main list.
5.  Click the character card to view and manage it.

## Features

*   **Character Creation and Management:** Create characters with names, classes, levels, ability scores, and hit points.
*   **HP and XP Tracking:** Easily track current hit points and experience points. The app shows progress to the next level.
*   **Leveling Up:** Automatically handles level-ups, allowing you to increase class levels or add new classes.
*   **Ability Score Increases:** Get reminders for ability score increases every four levels.
*   **Session-based XP:** Add experience points from your game sessions.
*   **Data Persistence:** All your character data is saved in your browser's local storage, so it persists between sessions.
*   **Delete Characters:** Remove characters you no longer need.
